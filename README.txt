WT-02

English

WT-02 is a IoT modular device which uses the ESP8266 SoC from Espressif. It runs NodeMCU, which enabled writing lightweight applications in Lua. It can collect data from a range of environment sensors and upload the results to a remote server.

Currently supported sensors include DS18B20, Si7021 and BMP280/BME280. The module is powered by a DC-DC step-down converter, with input voltage between 5.5V and 26V DC. The casing is designed to support up to 2 sensors at the same time, but the application does not support more than one sensor at the moment.

The software includes a captive portal, which helps the initial installation and configuration. The device can connect to a Wi-Fi network of your choice, assuming that it supports 802.11 b/g/n on 2.4 GHz. WPA-PSK and WPA2-PSK are supported.

Firmware installation is done using the UART port. You can use a ESP8266 flasher of your choice.

Application installation can be done using ESPlorer.

Bulgarian

WT-02 е модулно IoT устройство което използва ESP8266 чип от Espressif. Работи с NodeMCU, което позволява писането на леки приложения на езикът Lua. Може да събира данни от различни датчици за околна среда и да ги изпраща към отдалечен сървър.

Текущо се поддържат датчиците DS18B20, Si7021 и BMP280/BME280. Устройството се захранва от правотоков понижаващ конвертор с входящо напрежение от 5.5V до 26V DC. Корпуса е направен да поддържа до два датчика едновремено, но към момента приложението не поддържа повече от един.

Софтуера включва задържащ (captive) портал, който улеснява първоначалната инсталация и конфигурация. Устройството може да се свърже към Wi-Fi мрежа по Ваш избор, стига да поддържа стандартите 802.11 b/g/n на 2.4 GHz. Поддържаните стандарти за защита на мрежата са WPA-PSK и WPA2-PSK.

Инсталирането на операционната система на устройство се прави чрез серийния(UART) порт. Може да ползвате съвместим с чипа софтуер за програмиране по Ваш избор.

Инсталирането на приложението може да бъде направено чрез използването на ESPlorer.
